# Banco de Dados SQL

Projeto acadêmico de modelagem e implementação de banco de dados relacional utilizando MySQL.

## Recursos implementados

- Criação de banco de dados
- Criação de tabelas
- Chaves primárias
- Chaves estrangeiras
- Inserção de dados
- Consultas SQL

## Tecnologias

- SQL
- MySQL
- MySQL Workbench
